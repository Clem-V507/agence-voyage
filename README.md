## Exercice 1 : Modélisation du SI d’une agence de voyage 

### Étape 0 : Création du projet

Pour bien commencer le projet, nous avons créé un GitHub Repository ``agence-voyage`` accompagné de son GitHub Project ``Suivi des tâches``, sous forme de tableau Kanban, pour le suivi de l'avancement du projet.

### Étape 1 : Préparation du dictionnaire de données

À partir des règles de gestion prédéfinies, nous avons pu construire un tableau Excel (``Dictionnaire_donnees.xlsx``), appelé dictionnnaire de donnée, référençant les attributs, primary key (PK) et foreign key (FK) par entité, leur type de données et description, associés à ces règles.

### Étape 2 : Épurer un dictionnaire de données

Après analyse du dictionnaire de données, nous avons constater que celui-ci ne nécessitait pas d'être épuré. Aucune mise-à-jour du fichier n'a donc été effectué.
